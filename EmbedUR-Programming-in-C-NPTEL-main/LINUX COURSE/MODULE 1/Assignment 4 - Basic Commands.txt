2)How will you find out what is the use of the ps command.

Ans: To find out the use of ps(process status) command we can use 
    i) By using man command which acts as a System's Manual Pager and also used for documentation
        
	man ps

    ii) By using whatis command to show a precise information of a command 
        
	whatis ps
